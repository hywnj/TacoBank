# TacoBank
타코뱅크는 영수증 OCR 인식을 통해 품목별 정산 기능을 제공하는 오픈뱅킹 서비스 프로젝트입니다.<br>
<br>

## **📌 서비스 개요**
타코뱅크는 사용자가 보다 쉬운 금융 생활을 누렸으면 좋겠다는 생각에서 시작된 프로젝트입니다.
<br><br>

### **🔍 기획 의도**
> 오랜만에 지인들과 만나 술 한잔할 때, 누군가는 술을 안 마시거나 늦게 합류해서 각자 내야 할 금액이 달라지는 경우 있잖아요. <br>
> 이럴 때 직접 계산해서 나누기도 하고, 그냥 조용히 안 먹어도 같이 내기도 하고… 혹시 이런 문제로 다툰 적 없으신가요? <br>
> 그리고 다양한 금융 서비스를 이용하면서 계좌를 관리하고 거래하는 과정들이 번거롭지 않으신가요?

💡 **_타코뱅크는 이런 문제를 해결하기 위해 만들어졌습니다!_**
- **정산의 불편함 해소** : 반복적인 정산 요청, 개별 금액 계산, 직접 계좌 번호 입력 등 번거로움을 없앴어요!
- **통합 금융 정보 제공** : 여러 금융 계좌를 한곳에서 조회하고 관리할 수 있어요!
- **간편한 송금 프로세스** : 타 금융기관의 앱을 열 필요없이, 어떤 계좌든 바로 송금할 수 있어요!
<br><br>

## **✨ 주요 기능**
- **영수증 OCR 기반 정산**: 영수증을 기반으로 한 더치페이 기능 제공 <br>
    
  > 영수증을 업로드하면 OCR 모델이 항목별 내역을 추출해서, 각 사용자가 선택한 메뉴만 포함하여 계산할 수 있어요.<br>
  > 즉, `자신이 먹거나 포함된 항목에 대해서만 1/N(균등 분배)하여 정산하는 방식`이죠.<br>
  > 덕분에 술을 안 마신 사람이나 늦게 온 사람도 공정하게 계산할 수 있어, 불필요한 오해나 불만 없이 깔끔하게 정산할 수 있어요!
  
- **바로 송금**: 요청 받은 정산을 서비스내에서 바로 송금 가능 <br>
    
  > 요청받은 정산 내역에서 `단 한번의 버튼 클릭`과 비밀번호 입력만으로 정산자에게 송금할 수 있어요.<br>
  > 타 금융기관 앱을 열 필요 없이, 쉽고 빠르고 깔끔하게 정산할 수 있어요! <br>

- **오픈뱅킹 연동**: 계좌 관리 및 송금 기능<br>
    
  > 나의 모든 계좌정보를 `한 눈에` 확인하고, 여러군데의 금융기관 앱을 열 필요없이 `타코뱅크만 있어도` 송금할 수 있어요!<br>
  
- **본인 인증**: 문자로 본인 인증<br>
    
  > 금융 서비스의 핵심인 `보안`을 지키기 위해 최소한의 장치로 `문자 본인 인증`을 구현했어요!<br>
  > 물론, 실제 서비스에서는 더 강력한 본인 인증 방식을 구현해야하지만, 사이드 프로젝트로는 개인인증 관련 API를 사용할 수가 없어서<br>
  > 현실적인 범위에서 가장 간편하고 안전하다고 판단된 문자 인증으로 구현했습니다.
